Detailed Description

Contains the input/output data of the FreeTransactionManager::prepareComputeSignature( const std::shared_ptr<BasicSignatureVerificationData>) method for basic signature computation using the "Data Cipher" command.

An instance of this interface can be obtained via the method keypop::calypso::crypto::legacysam::LegacySamApiFactory::createBasicSignatureVerificationData().
