Detailed Description

Transaction manager dedicated to execute commands prepared asynchronously using AsyncTransactionCreatorManager::exportCommands().

An instance of this interface can be obtained via the method ::createAsyncTransactionExecutorManager( const std::shared_ptr<CardReader>, const LegacySam, const std::string&). AsyncTransactionCreatorManager 0.2.0
