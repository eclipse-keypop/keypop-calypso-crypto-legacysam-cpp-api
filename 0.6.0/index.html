Keypop Calypso Crypto Legacy SAM C++ API Documentation

Overview

The Keypop Calypso Crypto Legacy SAM C++ API defines interfaces for secure access module (SAM) cryptographic operations required for Calypso card transactions. This C++ interface provides port of the Calypso Networks Association specifications for legacy SAM operations.

Exception Handling

The API implements the following exception hierarchy:

  • keypop::calypso::crypto::legacysam::transaction::ReaderIOException Card/SAM reader communication failure handling
  • keypop::calypso::crypto::legacysam::transaction::SamIOException SAM communication failure handling
  • keypop::calypso::crypto::legacysam::transaction::SamRevokedException SAM revocation status handling
  • keypop::calypso::crypto::legacysam::transaction::UnexpectedCommandStatusException Unexpected SAM command response handling
